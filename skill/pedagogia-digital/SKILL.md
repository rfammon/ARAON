## Design Instrucional (REQUIRED)

Siga modelos consagrados para estruturar experiências de aprendizagem eficazes.

```markdown
**Modelo ADDIE:**
1. **Analyze (Analisar):** Identifique o público-alvo, objetivos de aprendizagem e lacunas de conhecimento.
2. **Design (Desenhar):** Estruture o conteúdo, defina estratégias de avaliação e selecione mídias.
3. **Develop (Desenvolver):** Crie os materiais didáticos, configure o LMS e produza recursos multimídia.
4. **Implement (Implementar):** Lance o curso, treine instrutores e suporte os alunos.
5. **Evaluate (Avaliar):** Realize avaliações formativas e somativas para melhoria contínua.

**Dica:** Para projetos ágeis, considere o modelo SAM (Successive Approximation Model).
```

## Metodologias Ativas

Priorize o aluno como protagonista.

- **Sala de Aula Invertida (Flipped Classroom):** Conteúdo teórico em casa (vídeos, leituras), prática e discussão em sala/síncrono.
- **Aprendizagem Baseada em Projetos (PBL):** Alunos aprendem resolvendo desafios reais.
- **Gamificação:** Use elementos de jogos (pontos, badges, narrativas) para aumentar o engajamento, não apenas "jogos".

## Tecnologias Educacionais e LMS

Ao configurar ou criar conteúdo para Ambientes Virtuais de Aprendizagem (LMS):

- **Moodle:** Foco em atividades colaborativas (Fóruns, Glossários, Wikis). Use rótulos para organização visual.
- **Canvas/Blackboard:** Utilize módulos sequenciais claros. Garanta design responsivo.
- **Acessibilidade (WCAG):**
  - Use textos alternativos (Alt Text) em imagens.
  - Garanta contraste de cor adequado.
  - Legende vídeos.

## Estrutura de Objetivos de Aprendizagem (Bloom)

Use a Taxonomia de Bloom revisada para definir objetivos claros:

1. **Lembrar:** Reconhecer, listar, descrever.
2. **Entender:** Interpretar, exemplificar, resumir.
3. **Aplicar:** Executar, implementar, usar.
4. **Analisar:** Diferenciar, organizar, atribuir.
5. **Avaliar:** Checar, criticar, julgar.
6. **Criar:** Gerar, planejar, produzir.
